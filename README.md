# CircleProgressTextView
CircleProgressTextView


## Use in help :

***

Step 1. Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

Step 2. Add the dependency

	dependencies {
			compile 'com.github.pdliugithub:CircleProgressTextView:v1.0'
	}
